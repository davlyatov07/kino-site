> Персональный веб-сервис для просмотра и управления коллекцией фильмов с Telegram ботом

---

## 📌 Описание проекта

**DYNЁ TV** — это полноценный веб-сервис (Hybrid chatbot) разработанный на Python + Django.  
Пользователь может искать фильмы через TMDB API, добавлять в избранное, оставлять отзывы и управлять своей коллекцией.  
Telegram бот работает 24/7 через Webhook и отвечает на команды пользователя.

## ⚙️ Инструкция по установке

### 1. Клонировать репозиторий
```bash
git clone https://github.com/davlyatov07/kino-site
cd kino-site
```

### 2. Установить зависимости
```bash
pip install -r requirements.txt
```

### 3. Применить миграции базы данных
```bash
python manage.py migrate
```

### 4. Создать администратора
```bash
python manage.py createsuperuser
```

---

## 🚀 Инструкция по запуску

```bash
python manage.py runserver
```

Открыть в браузере: `http://127.0.0.1:8000`

## 🌟 Основные функции сайта

### 👤 Авторизация
- Регистрация с подтверждением через Email (6-значный код)
- Вход и выход из аккаунта
- Профиль пользователя с аватаркой

### 🎬 Фильмы
- Главная страница с Hero-слайдером
- Каталог фильмов с карточками
- Поиск и фильтрация по жанру, году, рейтингу
- Страница фильма с описанием и трейлером
- Добавление через TMDB API автоматически

### ⭐ Личный кабинет
- Избранное
- Просмотренные фильмы
- Список "Хочу посмотреть"
- Отзывы и оценки

### 📊 Панель администратора
- Управление фильмами
- Графики по жанрам и годам
- Статистика коллекции
